# homebridge-google-nest-sdm

A homebridge plugin that uses the Google Smart Device Management API.

# Example Homebridge config:

    {
      "platform" : "homebridge-google-nest-sdm",
      "options": {
        "clientId": "...",
        "clientSecret": "...",
        "projectId": "...",
        "refreshToken": "..."
      }
    }
